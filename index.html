Mp3-Player

A simple Mp3 player project that is fully functional for music listening. Every user can browse through the songs he has added to the application. It features the common buttons like pausing, continuing, as well as skipping songs. In addition, each track displays the artist and title. Built with Java, JavaFX and Scene Builder.

Weather-App

A great project and amazing tool to use. Type a city and see real-time weather with icon-display, temperature, wind speed and humidity. Fully functional and precise using an API. Built with HTML, CSS & JS.
